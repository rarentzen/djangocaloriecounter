Welcome to Calorie Counter Help!



In order to utilize all the abilities of the Calorie Counter application, you must login.
If you do not already have an account please select the “Register” link at the top-right of the page.




Logging in:

Please enter your Username and Password. If you do not have an account already please select the “Register Here.”




Registration:

As you can see it is easy to register for a new account.
Simply fill out the required fields of Username, email, First Name, Last Name, Password and Confirm Password, and select the Register button.




Calorie Tracker Interface:

  • Here you can select from the drop-down Category menu either: Breakfast, Lunch, Dinner, or Snacks.
  • The food input box allows you to input the food you wish to enter. This field has a search attribute with
  • 15,000 available foods, so it is easy to find an available food.
  • The quantity input box allows you to add the quantity of food for your desired meal.
  • Select the “Add” button once you have entered in the desired food and quantity.



Nutrition History:

The nutrition history page can be found by selecting the nutrition history link from the top menu.
Once here you can view the history for a given day by entering the month, day, and year in mm/dd/yyyy format in the input box.




Logging Out:

Simply select Logout from the top menu and the application will log you out.
